# G2Ray Reality Panel (ضد فیلترینگ پیشرفته ایران)

پنل خودکار و رایگان ساخت کانفیگ **VLESS + XTLS + Reality** برای عبور از فیلترینگ.

## 🛡️ چرا این روش کار می‌کند؟
این پروژه به‌جای شبیه‌سازی سایت‌های خارجی، خود را به‌عنوان یک کاربر عادی **آپارات** جا می‌زند.
از آنجایی که سایت‌هایی مثل آپارات، دیجی‌کالا و ورزش۳ در ایران فیلتر نیستند، ترافیک شما برای سیستم‌های DPI کاملاً عادی و بی‌خطر به نظر می‌رسد.

## 📥 نحوه دریافت و استفاده از کانفیگ
1.  **دستی**: به تب Actions بروید، `G2Ray Reality Panel` را انتخاب و اجرا کنید.
2.  **خودکار**: هر ۴ ساعت یک کانفیگ جدید ساخته می‌شود.
3.  پس از اتمام، روی job کلیک کرده و لینک `vless://...` را از قسمت `Generate VLESS` کپی کنید.
4.  لینک را در کلاینت خود (v2rayN - Nekoray - v2rayNG) Import کنید.

## 🖥️ کلاینت‌های پیشنهادی
*   **Windows**: [Nekoray](https://github.com/MatsuriDayo/nekoray) یا [v2rayN](https://github.com/2dust/v2rayN)
*   **Android**: [v2rayNG](https://github.com/2dust/v2rayNG)
*   **iOS**: [FoXray](https://apps.apple.com/app/foxray/id6448898396)

## 💡 نکات مهم
*   **هزینه**: کاملاً رایگان روی سرورهای گیت‌هاب.
*   **IP**: هر بار اجرا یک IP جدید و تمیز دریافت می‌کنید.
*   **امنیت بالا**: با اثر انگشت کروم و SNI آپارات، ترافیک شما دقیقاً شبیه تماشای ویدئو در یک سایت داخلی است.

## 🚀 اجرای محلی
