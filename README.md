# G2Ray Reality Panel – ضد فیلترینگ پایدار

پنل **کاملاً رایگان** و خودکار برای ساخت کانفیگ `VLESS + XTLS + Reality` روی زیرساخت گیت‌هاب.

## ✨ چرا کار می‌کند؟
- شبیه‌سازی ترافیک **آپارات** یا سایت‌های محبوب ایرانی (دیجی‌کالا، ورزش۳...)
- پروتکل Reality بدون اثر انگشت
- تونل خودکار (Bore یا SSH) بدون نیاز به ثبت‌نام
- **هر ۴ ساعت یک IP جدید** و سرور تازه

## 🚀 روش گرفتن کانفیگ
1. این مخزن را **Fork** کنید.
2. **حتماً مخزن Fork شده را خصوصی (Private) کنید** تا لاگ‌های Actions برای عموم قابل مشاهده نباشند.
3. به تب **Actions** بروید و workflow `G2Ray Reality Panel` را اجرا کنید.
4. پس از ۱–۲ دقیقه، وارد job شوید و خروجی را ببینید.
5. در بخش **کانفیگ آماده**، لینک `vless://...` را کپی کرده و در کلاینت خود (v2rayN, Nekoray, v2rayNG) وارد کنید.

## ⏱️ وضعیت سرور
- سرور تا ۶ ساعت فعال می‌ماند.
- هر ۵ دقیقه یک پیام در لاگ چاپ می‌شود.
- با Schedule یک بار در هر ۴ ساعت خودکار ری‌استارت می‌شود.

## 🛡️ نکات امنیتی
- **مهم**: اگر Fork شما **عمومی** باشد، همه‌ی افراد لینک و کانفیگ شما را می‌بینند! حتماً Fork را **خصوصی** کنید.
- UUID و Public Key را جایی منتشر نکنید.
- در صورت فیلتر شدن تونل، یک اجرای دستی جدید انجام دهید یا از گزینه `--backup` استفاده کنید.

## 🔧 سفارشی‌سازی
- تغییر SNI:  
  `python3 g2ray_panel.py --sni "www.digikala.com,digikala.com"`
- انتخاب تصادفی از بین سایت‌های ایرانی:  
  `python3 g2ray_panel.py --backup`
